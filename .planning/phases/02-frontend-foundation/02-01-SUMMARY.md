## Decisions Made
- **fetch-to-canvas for MJPEG:** Mandatory for 24/7 kiosk. `<img src="/video_feed">` leaks memory at 15 FPS / 720p and crashes after 2-4 hours.
- **ws:// protocol detection:** `main.ts` constructs WebSocket URL using `window.location.protocol` to automatically use `wss://` on HTTPS, future-proofing for production.
- **setOnStateUpdate setter function:** TypeScript's `verbatimModuleSyntax` prevents re-exporting mutable bindings. A setter function pattern works cleanly.
- **Added frontend_dist/ to .gitignore:** Build artifacts should not be committed; regenerated by `npm run build`.
