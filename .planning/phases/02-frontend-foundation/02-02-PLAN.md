# Generate a smaller thumbnail for the event log (320px wide, lower quality)
event_snapshot = ""
if settings.WEBHOOK_INCLUDE_SNAPSHOT:
    event_snapshot = encode_snapshot_base64(frame, bbox=event.bbox, target_width=320, jpeg_quality=50)

Use target_width=320 and jpeg_quality=50 (smaller than webhook's 640/70) to keep the WebSocket payload reasonable. The event_log deque has maxlen=100, so worst case is ~100 thumbnails at ~5-10KB each = ~500KB-1MB total in the snapshot() payload. This is acceptable for a local WebSocket.

Do NOT modify the webhook snapshot (that stays at 640/70). Only the event_log push_event gets the smaller thumbnail.

Verify `encode_snapshot_base64` is imported at the top of main.py (it should already be imported since it's used on line 414).

   **updateStatusPanel(data: DashboardSnapshot): void**

     - Active tracks (Persoane active): `data.current_people`
     - Entries today (Intrari astazi): `data.entries_today`
     - Uptime (Timp functionare): format as `Xh Ym Zs` from `data.uptime_seconds`
     - Detector status: badge showing "Activ" (green) or "Oprit" (red) based on `data.detector_running`
     - Webhook status: badge showing "Functional" or "Eroare" based on `data.webhook_status.last_error`
     - Wake lock status: badge showing "Activ" or "Oprit" based on `data.wake_lock_active`

   - Use Romanian labels from inline constants (will be refactored to ro.ts in Plan 03).
   - Helper: `setText(id: string, text: string)` sets element.textContent
   - Helper: `setBadge(id: string, text: string, status: 'ok' | 'warn' | 'err')` sets text and CSS class

   **updateEntryLog(eventLog: EventLogEntry[], container: HTMLElement): void**
   - Track `lastEventCount` at module level
   - Compare `eventLog.length` to lastEventCount to detect new entries
   - For new entries (at the front of the array), create a table row with:
     - Timestamp: format ISO string to `HH:MM:SS`
     - Person ID: `data.person_id`
     - Confidence: `(data.confidence * 100).toFixed(0)%`
     - Snapshot: if `entry.snapshot` exists, render as `<img src="data:image/jpeg;base64,${entry.snapshot}" width="64" height="48">`, otherwise show a dash placeholder
   - Prepend new rows to the container (newest first)
   - Cap displayed rows at 100: remove excess from the end
   - Export a `resetEntryLog(): void` function that resets lastEventCount to 0 (for WebSocket reconnect full-rebuild)

   **formatUptime(seconds: number): string** — helper exported for reuse
   - Format as `Xh Ym Zs`

   **formatTime(isoString: string): string** — helper
   - Parse ISO 8601 timestamp and return `HH:MM:SS` in local time
