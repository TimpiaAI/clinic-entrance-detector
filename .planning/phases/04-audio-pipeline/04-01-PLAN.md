**2. Add apiTranscribe to api.ts:**

Add a new exported function (do NOT use the generic post helper -- FormData needs a different fetch call without Content-Type header):
```typescript
export async function apiTranscribe(
  audioBlob: Blob,
  initialPrompt?: string,
): Promise<TranscribeResult> {
  const form = new FormData();
  form.append('audio', audioBlob, 'recording.webm');
  if (initialPrompt) {
    form.append('initial_prompt', initialPrompt);
  }
  try {
    const res = await fetch('/api/transcribe', { method: 'POST', body: form });
    if (!res.ok) throw new Error(`Transcribe failed: ${res.status}`);
    return (await res.json()) as TranscribeResult;
  } catch (err) {
    console.error('api: POST /api/transcribe failed', err);
    throw err;
  }
}
```

Import TranscribeResult from types.ts at the top of api.ts.

**3. Create frontend/src/audio.ts:**

Create the full audio module with these exports:
- `checkMicAvailability(): boolean` -- checks `navigator.mediaDevices` existence (secure context guard). Returns false if not available.
- `initAudio(): Promise<boolean>` -- creates/resumes AudioContext inside user gesture, requests mic permission via getUserMedia (then immediately releases the test stream). Sets `micPermissionGranted = true` on success, returns boolean.
- `isMicReady(): boolean` -- returns `micPermissionGranted` state.
- `recordAndTranscribe(durationMs?: number, initialPrompt?: string): Promise<TranscribeResult>` -- the core function:
  1. Call `navigator.mediaDevices.getUserMedia({ audio: true })` to get stream
  2. Create `MediaRecorder` with `audio/webm;codecs=opus` MIME type (fall back to `audio/webm` if not supported via `isTypeSupported`)
  3. `recorder.start()` with NO timeslice (stop-and-collect pattern)
  4. `await new Promise(resolve => setTimeout(resolve, durationMs))` (default 10000ms)
  5. Stop recorder via Promise wrapping `onstop` event
  6. **CRITICAL:** Release mic tracks: `stream.getTracks().forEach(t => t.stop())` inside `onstop`
  7. Create `new Blob(chunks, { type: mimeType })`
  8. Call `apiTranscribe(blob, initialPrompt)` and return result

Module-level state:
- `let audioCtx: AudioContext | null = null` -- created once, reused
- `let micPermissionGranted = false`
- `const RECORD_DURATION_MS = 10_000`

Import `apiTranscribe` from `./api.ts` and `TranscribeResult` from `./types.ts`. Re-export `TranscribeResult` type for convenience.

  // Initialize audio on first Start press (user gesture required for mic + AudioContext)
