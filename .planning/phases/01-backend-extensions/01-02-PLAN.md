  <action>
    1. Create `api/transcribe.py` with:
       - `import re, tempfile, os`
       - `from fastapi import APIRouter, UploadFile, File`
       - `from faster_whisper import WhisperModel`
       - Module-level `_model: WhisperModel | None = None`
       - `get_model() -> WhisperModel` function: lazy-loads on first call using `WhisperModel(os.getenv("WHISPER_MODEL", "medium"), compute_type=os.getenv("WHISPER_COMPUTE_TYPE", "int8"))`. Stores in `_model` global. Returns existing on subsequent calls.
       - `extract_cnp(text: str) -> str | None`:
         - `digits = re.sub(r"[^0-9]", "", text)`
         - If len >= 13, return first 13
         - If len >= 10, return all digits (partial CNP for user confirmation)
         - Else return None
       - `extract_email(text: str) -> str | None`:
         - Port EXACTLY from controller.py (see RESEARCH.md "CNP/Email Extraction" code example)
         - Normalize: "punct/dot" + common TLDs -> "."
         - Normalize: "arond/arong/aroon/arun/arung/at/et/ad/a run/a rung" -> "@"
         - If "@" in result and "." in domain part, clean up spaces and non-email chars
         - Return cleaned email or None
       - `router = APIRouter(tags=["transcribe"])` with:
         - `POST /api/transcribe`: accepts `audio: UploadFile = File(...)`, saves to temp file (suffix based on content_type: .webm or .wav), calls `model.transcribe(tmp_path, language="ro", vad_filter=True)`, joins segments, extracts CNP and email, returns `{"text": text, "cnp": cnp, "email": email}`. Deletes temp file in finally block.

         - Verify response shape: {"text": str, "cnp": ..., "email": ...}

    3. Add env vars to `.env` file (append to end):


    CRITICAL: Do NOT shell out to ffmpeg. faster-whisper's PyAV handles WebM natively.
